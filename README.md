📌 Project Title

Data Visualization on Books Dataset

🎯 Objective

The goal of this project is to convert raw books data into meaningful visual insights using Python.

📂 Dataset

The dataset contains:

Book Title
Price
Rating
🛠 Tools Used
Python
Pandas
Matplotlib
Seaborn
📊 Visualizations Created

✔ Bar Chart – Book Price Comparison
✔ Histogram – Price Distribution
✔ Count Plot – Rating Distribution
✔ Scatter Plot – Price vs Rating
✔ Box Plot – Outlier Detection
✔ Heatmap – Correlation Analysis

📈 Key Insights
Book prices vary widely across titles
Most books are in mid-price range
Ratings are mostly between 3 and 5
No strong correlation between price and rating
Some outliers exist in pricing
🚀 How to Run This Project
1. Install required libraries
pip install pandas matplotlib seaborn
2. Run the Python file
python visualization.py
